# Python协同过滤电影推荐系统 python321

## 项目概述

本项目是基于Python语言的协同过滤算法实现的的电影推荐系统。系统利用用户历史评分数据，通过算法分析，为用户推荐可能喜欢的电影。

## 技术栈

- 后端：Django
- 数据库：SQLite
- 前端：HTML, CSS, JavaScript

## 功能模块

### 用户相关

- 电影搜索
- 电影收藏
- 电影评分

### 电影管理

- 电影信息展示
- 电影资讯发布
- 电影分类管理

### 系统管理

- 用户管理
- 电影评分管理

## 系统角色

- 普通用户：可以进行电影搜索、收藏和评分。
- 管理员：负责电影信息发布、用户管理和系统维护。

## 运行环境

- Python 3.*
- Django 3.*
- SQLite

## 部署步骤

1. 确保Python和Django环境已经搭建好。
2. 克隆或下载项目代码。
3. 安装项目依赖。

5. 运行Django应用。
